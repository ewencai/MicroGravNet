## 1. 软件概要

### 1.1 用途

微重力测网设计系统用于**微重力勘探测网的正向设计**，即在野外施工前，依据目标地质体的
物性与几何参数，定量确定：

- **扩边距离**——测区应向目标体边界之外延伸多远，以完整采集重力异常；
- **采样间隔**——在保证插值精度的前提下，各分区可采用的最大点距；
- **目标区/扩展区边界**——由目标区多边形扩展得到的实际施工边界；
- **测点布设**——渐变 / 分级 / 混合三种测网模式下的全部测点坐标。

系统将上述四步整合为一个 4-Tab 向导式 MATLAB App，支持分步执行、中间结果保存与工程续算。

### 1.2 适用范围

- 目标体可近似为**长方体、垂直圆柱体、水平圆柱体或球体**的微重力勘探；
- 目标区边界为任意**凸/凹多边形或矩形**；
- 埋深、尺寸、密度差已知或可估计的正演设计场景。

### 1.3 系统需求

| 项目 | 要求 |
|------|------|
| 运行环境 | MATLAB R2021a+（含 App Designer 运行时）或编译后的独立 EXE |

## 3. 技术原理与公式

### 3.1 重力正演模型

四种模型均计算观测面（z=0）上的**垂直重力异常 Δg**。异常单位经比例因子换算后统一以
mGal（或 g.u.）表示。密度差在代码中以 SI 单位（kg/m³）参与计算，界面输入为 g/cm³
（乘 10³ 转换）。

#### 3.1.1 长方体（Cuboid）

采用棱柱重力的解析积分（Nagy 公式）。设长方体沿三轴半边长确定 8 个角点
(ξ_i, η_j, ζ_k)，观测点 (x, y, z)，定义角点核函数：

```
R = sqrt((ξ-x)² + (η-y)² + (ζ-z)²)

GH(ξ,η,ζ) = (ξ-x)·ln((η-y)+R)
          + (η-y)·ln((ξ-x)+R)
          - (ζ-z)·atan[ (ξ-x)(η-y) / ((ζ-z)·R) ]
```

重力异常为 8 个角点核函数的符号交替求和：

```
Δg = -G·ρ·1e8 · Σ_{i,j,k=1,2} (-1)^{i+j+k} · GH(ξ_i, η_j, ζ_k)
```

其中角点由中心 (X0,Y0,Z0) 与尺寸给出：ξ = X0 ± L/2，η = Y0 ± W/2，ζ = Z0 ± H/2。
（实现：`CalcuGra_Cuboid.m` + `GH.m`）

#### 3.1.2 垂直圆柱体（Vertical Cylinder）

由二维物质线正演积分导出，柱轴竖直。设柱顶埋深 H_top、柱底埋深 H_base、
中心埋深 H_mid = D，观测点到柱轴水平距离为 s，令 θ = atan(s / H_mid)：

```
Δg = 2π·G·ρ · [ H_base − H_top
               + sqrt(R² + H_top²)
               − sqrt(R² + H_base²) ] · 1e8 · cos³θ
```

`cos³θ` 为观测点偏离柱轴的几何衰减因子。（实现：`CalcuGra_VerticalCylinder.m`）

#### 3.1.3 水平圆柱体（Horizontal Cylinder）

近似为有限长水平质量线。设线密度 λ = π·R²·ρ，柱中心 (X0,Y0,Z)，半长 HalfL，
观测点到柱中心的水平+垂直距离平方：

```
d² = (Z − z_obs)² + (X0 − x)² + (Y0 − y)²

Vz = 2·G·λ·|Z − z_obs|·HalfL / ( d² · sqrt(d² + HalfL²) ) · 1e5
```

（实现：`CalcuGra_HorizontalCylinder.m`）

#### 3.1.4 球体（Sphere）

等效点质量 M = (4/3)π·R³·ρ。设观测点到球心矢量 (dx, dy, dz)，r = |(dx,dy,dz)|：

```
Vz = G·M·(−dz) / r³ · 1e5
```

系统同时可输出全张量梯度分量（供扩展分析）：

```
Vzz = G·M·(3dz² − r²)/r⁵ · 1e9      Vxx = G·M·(3dx² − r²)/r⁵ · 1e9
Vyy = G·M·(3dy² − r²)/r⁵ · 1e9      Vxy = G·M·(3dx·dy)/r⁵ · 1e9
Vxz = G·M·(3dx·(−dz))/r⁵ · 1e9      Vyz = G·M·(3dy·(−dz))/r⁵ · 1e9
```

（实现：`CalcuGra_Sphere.m`）

### 3.2 扩边距离确定原理

原理：目标体产生的重力异常在其边界外仍有"拖尾"。为完整采集异常，测区须向外
扩展至异常衰减到峰值的某一比例（阈值）以下。

1. 在对称测网上正演得到重力异常网格 Gra，取过中心的中剖面右半段 prof；
2. 归一化：`Gnorm = prof / max(Gra)`，峰值处为 1；
3. 沿距离方向找到 Gnorm 首次降至阈值 thr 以下的位置，线性插值求交点距离：

```
ExtDistance = x1 + (thr − y1)·(x2 − x1)/(y2 − y1)
DepthRatio  = ExtDistance / Depth
```

默认阈值 thr = 0.089（8.9%），此时对典型模型 `DepthRatio ≈ 2`，即扩边距离≈2×埋深，
这也是后续测网设计中"扩展终止于 2×MaxDepth"的依据。
（实现：`ComputeBoundaryProfile.m`）

### 3.3 采样间隔与 Kriging 误差分析

原理：以极密采样正演场为"真值"，对不同采样间隔的稀疏采样做 Kriging 插值，
比较插值场与真值场的相对误差，反推各分区可接受的最大间隔。

1. **理论真值场**：以用户指定的理想密集间隔正演，作为参考 `idealGrid`；
2. **多间隔采样 + Kriging**：对每个测试间隔 Δxᵢ 稀疏采样→散点 `.dat`→Surfer
   `GridData6` 克里金插值→网格 `Dk`；
3. **相对误差**：

```
relErr = |Dk − idealGrid| / (|idealGrid| + eps)
```

   取中剖面右半段构成误差矩阵 `ErrorMatrix`（行=采样间隔，列=距中心距离）；
4. **分区推荐**：按距离划分三区（相对埋深 D）：

```
核心区: 0 ≤ dist < 0.618·D
目标区: 0.618·D ≤ dist < 1.5·D
扩展区: dist ≥ 1.5·D
```

   各区取"最大误差 ≤ 阈值"的**最大**采样间隔作为推荐值（尽量贴近阈值等值线）。
   界面另绘制阈值等值线的紧支撑拟合直线，便于连续读取任意距离处的推荐间隔。
   （实现：`IntervalErrorAnalysis.m`；Surfer 封装：`SurferKrigingBatch.m`）

### 3.4 测网设计算法

三种模式共享以下基础构件。

#### 3.4.1 目标区规则网格与旋转

在目标区外接范围生成步长 Δx 的规则栅格，绕形心（或锚点）旋转角 θ：

```
Rm = [cosθ  −sinθ;
      sinθ   cosθ]
P' = (P − centroid)·Rm + centroid
```

保留落在目标区多边形内部（`inpolygon`）的点作为核心测点。
**锚点机制**：若指定锚点，则栅格以锚点为网格节点对齐，便于多期测网套合。
（实现：`gridInPolygon.m`）

#### 3.4.2 近边界点内缩

落在边界外但距边界小于 `NearFactor×Δx` 的候选点，沿指向形心方向内缩
`(1−RelaxFactor)×Δx`，避免测点恰好压在边界上或过于稀疏：

```
dir = (centroid − p)/|centroid − p| · (1−RelaxFactor)·Δx
p_new = p + dir
```

（实现：`gridInPolygon.m` + `pointToSegmentDistance.m`）

#### 3.4.3 边界扩展

- **矩形**（四点且通过 `IsRectangle` 判定）：对极值坐标直接 ±d，保持矩形；
- **不规则多边形**：每条边沿外法向平移 d，相邻平移边求**参数交点**得到新顶点。
  采用参数式而非斜率截距式，对垂直边稳健（避免除零）：

```
边方向 d1、d2，起点 p1、p2
解 A·t = (p2 − p1)，A = [d1(1) −d2(1); d1(2) −d2(2)]
若 |det(A)| ≈ 0（平行）取中点，否则交点 = p1 + t(1)·d1
```

（实现：`PolygonExtension.m` + `ExtendPolygonDistance.m` + `IsRectangle.m`）

#### 3.4.4 自适应周长采样

渐变/混合模式的外环沿闭合边界等弧长布点。为使点距接近目标值且首尾闭合无缝，
在 `baseInterval×(1±InterFactor)` 范围内选取能**整分周长**的间隔：

```
候选间隔 = perimeter / n，  n ∈ [floor(perim/max), ceil(perim/min)]
取落在允许区间且最接近 baseInterval 者
沿累计弧长 linspace 均匀插值布点
```

（实现：`AdaptivePerimeterSampling.m`）

#### 3.4.5 渐变测网（Gradient）

目标区均匀采样，向外分"扩展阶段"逐级放大点距，每阶段含若干圈：

```
第 stage 阶段步长: step(stage) = ExpandFactor^stage · Δx_inner
累计扩距 totalExt 每圈递增 step，直至 totalExt = 2·MaxDepth（封顶）
各圈以 polybuffer 圆滑扩展，周长自适应布点
最终以 2·MaxDepth 圆滑边界裁剪越界点
```

（实现：`DesignGradientSurveyNet.m`）

#### 3.4.6 分级测网（Graded）

目标区固定点距，外围划分 2~5 级环带，每级**独立**从目标区边界起算扩距、
使用各自的采样间隔填充**规则**网格（矩形保持矩形）：

```
第 g 级扩距: extDist(g) = gradeExtMults(g) · MaxDepth   （封顶 2·MaxDepth）
第 g 级点距: gradeIntervals(g)
环带 = 第 g 级外边界 PolygonExtension \ 第 g−1 级边界，规则网格填充
```

（实现：`DesignGradedSurveyNet.m` + `gridInAnnulus.m`）

#### 3.4.7 混合测网（Hybrid）

内部分级 + 外围渐变，兼顾形状保真与平滑衔接：

- 内部环带（除最外级）用 `PolygonExtension` 规则扩展，保留形状特征；
- 最外一级分级环带改用 `polybuffer` 圆滑边界，与渐变区平滑过渡；
- 渐变区从最外分级环带向外，**以最外级点距为基础间隔**做周长自适应采样；
- 最终以 2×MaxDepth 圆滑边界裁剪。

（实现：`DesignHybridSurveyNet.m`）

## 5. 操作步骤

### 5.1 Tab1 — 扩边距离确定

1. 选择模型类型，填入密度差、尺寸、埋深；
2. 设置正演采样间距与阈值（默认 0.089）；
3. 点击「计算扩边距离」——上轴显示 3D 模型示意，下轴显示归一化重力异常剖面+阈值线；
4. 读取标注的扩边距离与埋深倍数；
5. 点击「采用此结果」将扩边距离与模型参数传递给 Tab2/Tab3。

### 5.2 Tab2 — 采样间隔确定

1. **Step 2.1** 沿用 Tab1 模型，设理想采样间隔与测试间隔范围，点击「生成理论场」；
2. **Step 2.2** 点击「Kriging 插值」（需 Surfer，后台静默运行），批量插值各间隔；
3. **Step 2.3** 设误差阈值，点击「计算误差矩阵」，绘制相对误差云图；
4. **Step 2.4** 查看各区推荐间隔与阈值等值线拟合线，可手动微调；
5. 导出误差云图 PNG、误差矩阵 ASC/CSV、分区推荐表 CSV。

> 无 Surfer 时仅可完成 Step 2.1 生成理论场。

### 5.3 Tab3 — 目标区确认

1. 「加载边界文件」，每行一个顶点坐标（制表符/逗号分隔）；
2. 填扩边距离（默认继承 Tab1）；
3. 「计算扩展边界」——自动判定矩形/不规则多边形并采用相应扩展算法；
4. 查看目标区（红）+ 扩展区（蓝），可导出边界文件。

### 5.4 Tab4 — 测网设计

1. 选择设计模式：渐变 / 分级 / 混合；
2. 填写通用参数及所选模式的专属参数；
3. 点击「生成测网」——右侧显示分层着色测点图与统计摘要；
4. 导出「全部测点 (Excel)」与「设计图 (PNG)」。

### 5.5 项目保存

菜单「文件 → 保存」将全部 Tab 结果存为 `.mat`；「打开」可恢复续算。
输入框值自动跨会话记忆。

| 现象 | 原因 | 处理 |
|------|------|------|
| Tab2「Kriging 插值」报 COM 错误 | 未装 Surfer 或版本过低 | 安装 Surfer 13+；仅需理论场可跳过 |
| 首次 Kriging 首个文件失败 | Surfer COM 初始化未完成 | 已内置 pause(3) 等待；重试即可 |
| 扩展边界自交/畸形 | 凹多边形扩距过大 | 减小扩距，或检查边界顶点顺序 |
| 矩形扩展后变形 | 顶点非严格矩形 | 确认四点且直角，触发矩形分支 |
| 测点压在边界上 | 内缩比例过大/近边界因子不当 | 调 NearFactor / RelaxFactor |
| 授权对话框反复弹出 | License.key 缺失或过期 | 重新激活；套件内任一 App 激活即通用 |
