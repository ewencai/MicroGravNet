## 概述

微重力测网设计系统是一个 4 步向导式 MATLAB 应用，用于微重力勘探中的测网设计。工作流依次为：

1. **扩边距离确定** — 正演模型计算重力异常，阈值分析确定扩距
2. **采样间隔确定** — Kriging 插值误差分析，确定各区最优采样间隔
3. **目标区确认** — 加载边界，计算扩展边界
4. **测网生成** — 渐变 / 分级 / 混合 三种模式

## 界面布局

- 顶部 4 个 Tab：扩边距离 | 采样间隔 | 目标区 | 测网设计
- 左侧 控制面板，右侧 可视化区域
- 底部状态栏，菜单栏（文件/帮助）
- 所有输入框值跨会话自动记忆

## 分步操作

### Tab 1 — 扩边距离确定

通过正演模型计算重力异常曲线，根据阈值确定扩边距离。

1. 选择模型类型（长方体/垂直圆柱体/水平圆柱体/球体）
2. 填入模型参数（密度差、尺寸、埋深）
3. 设置采样间距和阈值（默认 0.089 = 8.9%）
4. 点击「计算扩边距离」
5. 上轴：3D 模型示意图；下轴：归一化重力异常剖面 + 阈值线
6. 点击「采用此结果」传递给 Tab 3

### Tab 2 — 采样间隔确定

通过克里金插值误差分析确定各区最优采样间隔。

1. **Step 2.1**：沿用 Tab 1 模型，设理想采样间隔和测试间隔范围，点击「生成理论场」
2. **Step 2.2**：点击「Kriging 插值」（需 Surfer），批量进行多间隔克里金插值
3. **Step 2.3**：设误差阈值，点击「计算误差矩阵」
4. **Step 2.4**：在误差云图上查看推荐拟合线；可手动调整斜率/截距/扩展因子

可导出：误差云图 PNG、误差矩阵 ASC/CSV、分区推荐间隔表 CSV。

**注意**：Kriging 需要 Golden Software Surfer 13+。无 Surfer 时仅可生成理论场。

### Tab 3 — 目标区确认

1. 「加载边界文件」— 支持 .dat/.txt/.xlsx，每行一个顶点坐标（制表符或逗号分隔）
2. 填扩边距离（默认从 Tab 1 继承，或手动输入 = 2×最大埋深）
3. 「计算扩展边界」
   - 自动检测矩形/不规则多边形，使用相应扩展算法
   - 矩形边界保持矩形扩展，不规则多边形用边法向求交
4. 可视化：目标区（红）+ 扩展区（蓝）
5. 可「导出边界文件」

### Tab 4 — 测网设计

1. 选择设计模式：**渐变测网** / **分级测网** / **混合测网**

**通用参数**：
| 参数 | 说明 |
|------|------|
| 目标区点距 (m) | 核心区采样间隔 |
| 最大埋深 (m) | 扩展终止于 2× 此值 |
| 旋转角度 (°) | 测网旋转角 |
| 使用锚点 | 锚定某点为网格节点，对齐测线 |
| 近边界因子 | 判定"边界附近"的距离（×点距） |
| 内缩比例 | 边界附近点内缩比例 (0-1) |

**分级/混合专属**：
| 参数 | 说明 |
|------|------|
| 分级数 (2–5) | 外扩环带级数 |
| 各级间距 (m) | 逗号分隔：第1个=目标区，后续=逐级环带 |
| 各级扩边倍数 (×埋深) | 对应每级从目标区边界起算的扩距 |

**渐变/混合专属**：
| 参数 | 说明 |
|------|------|
| 扩距因子 (>1) | 逐阶段间隔扩大倍数 |
| 每阶段圈数 | 每个 Expansion Stage 内的圈数 |

2. 点击「生成测网」
3. 导出：「导出全部测点 (Excel)」「导出设计图 (PNG)」

**混合测网设计逻辑**：
- 内部环带（除最外环）使用规则边界（`PolygonExtension`，保留形状特征）
- 最外环使用圆滑边界（`polybuffer`），与渐变区平滑衔接
- 渐变区从最外环边界继续向外，使用周长自适应采样
- 最终以 2×最大埋深圆滑边界裁剪

## 输出文件格式

**测点 Excel**：列 `(X坐标, Y坐标, 线号, 点号, 点名称)`

| 区域 | 命名格式 | 示例 |
|------|----------|------|
| 目标区 | `PPPCMMM` | `02PCM45`（第 02 点、第 45 线） |
| 扩展区 | `RRPCKNN` | `01PCK05`（第 01 环带、第 05 点） |

**测网设计图**：含边界、测点分层着色、比例尺、指北针、图例。

## 项目保存

菜单栏「文件 → 保存」将当前全部 Tab 结果存为 .mat 文件。下次「打开」可恢复。输入框值自动跨会话记忆。
