# MicroGravNet 安装程序说明

> 本文面向**最终用户**，介绍如何在没有 MATLAB 的电脑上，通过安装程序安装并运行
> MicroGravNet（微重力测网设计系统）v1.0.1。开发者环境搭建与源码编译请参见
> [安装手册](installation_guide.md)。

---

## 1. 交付物构成

分发包内通常包含以下文件（以实际交付为准）：

| 文件 | 说明 |
|------|------|
| `MyAppInstaller_web.exe` | **联机安装程序**，运行时自动下载 MATLAB Runtime（体积小） |
| `MyAppInstaller_mcr.exe` | **脱机安装程序**，已内置 MATLAB Runtime（体积大，无需联网） |
| `MicroGravNet.exe` | 主程序（若单独提供，需另装匹配版本的 Runtime） |
| `License.key`（可选） | 授权文件，由开发者签发 |
| `docs/` | 使用手册与说明书 |

> 通常只需运行其中一个安装程序（联机版或脱机版），二选一即可。

---

## 2. 运行环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10 / 11（64 位） |
| 磁盘空间 | 约 2–4 GB（含 MATLAB Runtime） |
| 内存 | 建议 8 GB 及以上 |
| MATLAB Runtime | 与编译所用 MATLAB 版本对应（安装程序会自动处理） |
| Golden Software Surfer | 13+（仅采样间隔 Kriging 功能需要，可选） |
| 权限 | 安装需管理员权限 |

**MATLAB Runtime 是免费的**，无需 MATLAB 许可证。它是运行独立可执行文件所需的
运行库，由安装程序自动安装；若单独拿到 `MicroGravNet.exe`，可从 MathWorks 官网
获取对应版本 Runtime：<https://www.mathworks.com/products/compiler/matlab-runtime.html>

---

## 3. 安装步骤

### 3.1 使用安装程序（推荐）

1. **右键**以管理员身份运行 `MyAppInstaller_web.exe`（或脱机版 `_mcr.exe`）；
2. 在欢迎界面点击「Next」；
3. **选择安装目录**（建议路径不含中文/空格，例如 `D:\MicroGravNet`）；
4. **选择 MATLAB Runtime 安装目录**（联机版将自动下载安装，需保持联网；
   脱机版直接从安装包安装）；
5. 阅读并接受许可协议，点击「Install」；
6. 等待安装完成（首次安装 Runtime 耗时较长，属正常现象）；
7. 点击「Finish」结束。

安装完成后，开始菜单与安装目录下会出现 **MicroGravNet** 快捷方式。

### 3.2 直接使用可执行文件（已装 Runtime 时）

若目标机器已安装匹配版本的 MATLAB Runtime，可将 `MicroGravNet.exe` 连同其
附属文件夹直接拷贝到本地，双击运行，无需安装程序。

> 首次启动会解压运行时组件，可能等待 10–60 秒，请耐心等待。

---

## 4. 首次启动与授权

1. 双击 **MicroGravNet** 快捷方式启动；
2. 若本机尚未激活，弹出**软件授权**对话框，显示本机**机器码**；
3. 将机器码提供给软件供应方，获取对应的 `License.key`；
4. 打开 `License.key`，**复制其全部内容**，粘贴到授权对话框，点击「激活」；
5. 激活成功后自动进入主界面，授权信息保存在
   `%APPDATA%\AppLicenseSystem\License.key`，后续启动免验证。

> **套件级共享**：本机若已为同套件内其他软件激活过，MicroGravNet 直接免激活通过。
> 授权采用 RSA-2048 签名，离线可用，无需联网校验。

---

## 5. Surfer 配置（可选）

「采样间隔确定」中的 Kriging 插值通过 COM 调用 Golden Software Surfer：

1. 安装 Surfer 13 或更高版本（安装时默认注册 COM 组件）；
2. 首次调用会后台静默启动 Surfer，无需手动打开；
3. 未安装 Surfer 时，其余功能（扩边距离、目标区、测网设计）均不受影响，
   「采样间隔」页仅能生成理论场。

---

## 6. 验证安装

1. 启动后确认主窗口标题为 `微重力测网设计系统 (MicroGravNet) v1.0.1`，含 4 个标签页；
2. 「③ 目标区确认」→「加载边界文件」→ 选一份 `.dat` 边界；
3. 填扩边距离 → 「计算扩展边界」，右侧显示目标区+扩展区；
4. 「④ 测网设计」→ 选「混合测网」→ 填参数 → 「生成测网」，显示设计图与总点数。

以上通过即安装成功。

---

## 7. 卸载

- **控制面板** → 「程序和功能」→ 选择 **MicroGravNet** → 卸载；或
- 运行安装目录下的卸载程序（若提供）。

> MATLAB Runtime 可独立保留（供其他 MATLAB 编译程序使用），也可在
> 「程序和功能」中单独卸载 **MATLAB Runtime**。授权文件位于
> `%APPDATA%\AppLicenseSystem\`，如需彻底清除可手动删除该目录。

## 8. 常见问题

| 现象 | 原因与处理 |
|------|-----------|
| 双击无反应/闪退 | Runtime 版本不匹配或未安装；用安装程序重装，或安装对应版本 Runtime |
| 启动很慢 | 首次启动解压组件属正常；后续启动会加快 |
| 反复弹出授权框 | `License.key` 缺失/无效；重新粘贴激活；确认系统时间正确 |
| 采样间隔页报 Surfer 错误 | 未装 Surfer 或 COM 未注册；安装 Surfer 13+ 后重试 |
| 导出 Excel/图片失败 | 目标目录无写入权限或磁盘已满；换目录或清理空间 |
| 安装卡在下载 Runtime | 联机版需联网；网络不稳时改用脱机版 `_mcr.exe` |
| 提示缺少 DLL | 安装 [Microsoft Visual C++ 运行库](https://learn.microsoft.com/cpp/windows/latest-supported-vc-redist) |

## 附：开发者如何生成安装程序

在装有 **MATLAB Compiler** 的机器上，由 `MicroGravNet.exe` 生成带 Runtime 的安装包：

**方式一 · Application Compiler（图形界面）**

1. MATLAB 命令行运行 `applicationCompiler`；
2. 主文件选 `main.m`，附加 `lib/`、`LicenseManager.m`；
3. 图标选 `resources/icon_mono.png`，启动画面选 `resources/splash_screen.png`；
4. Runtime 选「Runtime downloaded from web」(联机版) 或「Runtime included in package」(脱机版)；
5. 点击「Package」，输出 `MyAppInstaller_web.exe` / `_mcr.exe`。

**方式二 · 命令行**

```matlab
% 先编译独立程序
cd project        % 或 deploy
compile           % 生成 output/MicroGravNet.exe

% 再打包为安装程序（R2020b+）
compiler.package.installer( ...
    'output\MicroGravNet.exe', ...
    'output\requiredMCRProducts.txt', ...
    'ApplicationName', 'MicroGravNet', ...
    'AuthorName', '<作者>', ...
    'Version', '1.0.1', ...
    'RuntimeDelivery', 'web');   % 或 'installer' 生成脱机版
```

生成的安装程序即可分发给无 MATLAB 的最终用户。
